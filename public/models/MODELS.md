# Bundled 3D models

- plane.glb — "Cesium_Air" sample aircraft from the CesiumGS/cesium repository
  (https://github.com/CesiumGS/cesium, Apache License 2.0). Used as the ambient
  airliner, tinted at runtime with the active template's dominant colour.

## Optional (user-supplied — not bundled)

- starship.glb — a SpaceX Starship model. Free ones exist on Sketchfab/CGTrader
  but require an account to download, so none is bundled. Drop a GLB here and
  the SpaceX pad watcher (Starbase / LC-39A) activates automatically.
- tower.glb — optional Mechazilla / launch-tower model, same deal.
